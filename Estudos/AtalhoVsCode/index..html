Se é burro

  1. Alt + Shift + R: Organizar código
  2. Alt + ↑ ou ↓: Carregar linha
  3. Ctrl + Enter: Pula para linha de baixo ↓
  4. Ctrl + Shift + Enter: Pula para linha de cima ↑
  5. Shift + Alt + ↑ ou ↓: Copia a linha respectiva
  6. Ctrl + D: Altera códigos iguais
  7. Shift + Alt + A: No código, fica em comentário a parte selecionada
  8. li * x: Dentro da identação, vai em todos os termos
